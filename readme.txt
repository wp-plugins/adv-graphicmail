Add in your Wordpress site a subscribe form for your mailing list in GraphicMail.

== Description ==

This plugin allows you to add in your Wordpress site a subscribe form for your mailing list in your GraphicMail account.

*   After plugin activation, configure the API credentials in Settings > GraphicMail
*   In Appearance > Widgets find GraphicMail and drag it into your favorite widget area
*   Insert the title to show and select your preferred mailing list for emails subscription
*   Enjoy!

== Installation ==

1. Upload the entire directory `adv-graphicmail` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure the API credentials in Settings > GraphicMail

== Frequently Asked Questions ==

= I configured my account credentials in Settings > GraphicMail but after saving I got an error message. Which can be the problem? =

Make sure that your host IP address is allowed to access GraphicMail APIs.
In your GraphicMail account you have to add the host IP address to the list of trusted IP addresses in Advanced Settings > API Settings
